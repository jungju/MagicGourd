# Proposals

## 1. Support-Point Witness Barks
- **Problem:** Rotating support points change location, but the village rarely comments on that change.
- **Goal:** Place 2-3 tiny witness interactions near the support loop so players hear short reactions tied to branch + active point + tier.
- **Player feel:** The village notices where help is flowing.
- **Implementation difficulty:** Low-Medium.

## 2. Route Cue Prop Pass
- **Problem:** Players may rely mostly on HUD hint text to find the active point.
- **Goal:** Add tiny branch-colored rope/cloth/produce trail accents that subtly guide the current rotation route.
- **Player feel:** The path between repeatable errands reads more authored.
- **Implementation difficulty:** Low.

## 3. Tier-Triggered Micro Moments
- **Problem:** Tier upgrades are environmental, but their transition can still feel easy to miss.
- **Goal:** On first crossing Tier 1 / Tier 2, trigger one short local dialogue/effect beat.
- **Player feel:** Milestones land with a little more ceremony.
- **Implementation difficulty:** Medium.

## 4. Brother Reaction Expansion
- **Problem:** Heungbu and Nolbu react to endings, but not enough to the player's repeated post-ending labor.
- **Goal:** Add line sets that react to support-point rotation and landmark tiers.
- **Player feel:** The core brothers keep noticing the player's ongoing choices.
- **Implementation difficulty:** Low.

## 5. Passive Village Ledger / Gratitude Board
- **Problem:** Counters exist, but they are distributed across several signs.
- **Goal:** Add one lightweight board that summarizes recent communal drops or counted dues.
- **Player feel:** Repetition leaves a clearer village memory.
- **Implementation difficulty:** Medium.

## 6. Tiny Delivery Variants
- **Problem:** Hand-ins still always consume exactly one normal gourd in the same way.
- **Goal:** Add occasional accepted variants (seed return, herb add-on, inspection variant) without changing the main economy.
- **Player feel:** Post-ending farming has a bit more texture.
- **Implementation difficulty:** Medium-High.
