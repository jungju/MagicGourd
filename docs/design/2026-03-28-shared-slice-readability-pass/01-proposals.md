# Proposals

## Refine Targets Collected
1. Shared-lane prompt ownership when two players approach nearby swallows or plaza pads
2. Local-vs-remote popup dominance when another player's rewards fire in the same camera slice
3. Center-lane ownership when your planted pumpkin sits near another player's route or action
4. Retreat readability when goblin pressure and another player's movement occupy the same danger corridor
5. Billboard tolerance under multiplayer overlap so support signs do not become the third or fourth competing layer

## Candidate Set

### Candidate A — Shared prompt ownership contract
Define how nearby prompts should remain readable when other players are interacting in the same slice.

### Candidate B — Local-first feedback dominance contract
Define how reward/danger feedback should prioritize the local player's action beats over remote player noise without inventing a new UI layer.

### Candidate C — Combined shared-slice readability contract
Combine prompt ownership, local-first feedback, center-lane overlap, and danger retreat rules into one multiplayer finishing pass.

### Candidate D — Multiplayer-only Studio validation checklist
Do not add new design rules; only add a test checklist for 2-player overlap scenes.
