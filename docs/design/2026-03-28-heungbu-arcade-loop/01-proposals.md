# Proposals

## Proposal A
- Summary: Hard-cut the game into one shared arcade loop with owned pumpkins and three prompt upgrades.
- Why it helps: Removes ambiguity immediately and makes the first session readable.
- Cost: Requires replacing most of the current server/client flow.
- Risk: Some old story flavor becomes archival instead of live gameplay.

## Proposal B
- Summary: Keep the old story game and bolt an arcade sub-loop onto it.
- Why it helps: Lower initial rewrite cost.
- Cost: Leaves two competing product directions in one project.
- Risk: The game stays confusing and over-scoped.

## Proposal C
- Summary: Keep the story structure but shorten the crop and swallow steps.
- Why it helps: Preserves more authored narrative.
- Cost: The quest stack still dominates the multiplayer feel.
- Risk: Complexity remains even if timers shrink.
