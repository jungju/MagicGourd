# Candidate Set

## Candidate A — Ambient Ending Mood Layer
A lightweight world-state layer that changes patch text, yard signage, certain prop colors/materials, and key NPC ambient lines based on `endingChoice`.

**Direction:** UX / environmental storytelling

## Candidate B — Ending-Specific Free-Play Tasks
After `COMPLETE`, unlock one repeatable action route per ending:
- restoration: shared-harvest delivery
- tribute: toll/tribute delivery

**Direction:** systems / economy / replay

## Candidate C — Conflict Echo Layer
Keep Nolbu vs Heungbu tension alive through ambient pressure beats after completion, such as stricter gate messaging, patch inspections, or soft warnings, without reopening the main questline.

**Direction:** story / atmosphere

## Candidate D — Branch-Aware Ambient Dialogue Pack
Expand Heungbu/Nolbu/focal prompts with more branch-aware free-play lines and micro-responses while leaving the world props mostly unchanged.

**Direction:** narrative / low-cost polish

## Candidate E — Ending Landmark Prop
Spawn a visible symbolic prop or banner set reflecting the chosen ending and use it as the center of post-ending flavor.

**Direction:** visual payoff / staging
