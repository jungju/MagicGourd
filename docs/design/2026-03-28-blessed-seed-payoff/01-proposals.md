# Proposals

## Proposal A
- Summary: Add a blessed seed that grows a true magic gourd, then pays out through Heungbu with a simple retaliation follow-up.
- Why it helps: Directly advances the main tale with clear player-visible reward.
- Cost: Medium.
- Risk: Low; mostly extends existing quest and patch systems.

## Proposal B
- Summary: Make the magic payoff a cinematic chest-spawn event with multiple spawned rewards and ambient world changes.
- Why it helps: Strong spectacle.
- Cost: Medium-high.
- Risk: Higher chance of brittle Roblox presentation work without Studio validation.

## Proposal C
- Summary: Skip the blessed seed and jump straight to Nolbu sabotage using only dialogue.
- Why it helps: Fastest route to more conflict.
- Cost: Low.
- Risk: Undercuts the iconic payoff and weakens emotional rhythm.
