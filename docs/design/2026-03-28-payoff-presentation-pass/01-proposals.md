# Proposals

## 1. Reward Ceremony Presentation Pass
- **Problem:** The magic gourd climax lands mechanically but not emotionally.
- **Goal:** Add a short layered reward presentation using current HUD/effect primitives.
- **Player feel:** "I caused something special to happen."
- **Difficulty:** Low-Medium.

## 2. Nolbu Jealousy Pressure Pass
- **Problem:** Nolbu retaliation exists, but the threat spike is brief.
- **Goal:** Make retaliation feel more hostile through staged warning beats and clearer recovery framing.
- **Player feel:** "My kindness changed the village, and Nolbu noticed."
- **Difficulty:** Medium.

## 3. Patch Recovery Feedback Pass
- **Problem:** Clearing brambles resolves the quest, but the restoration beat is visually thin.
- **Goal:** Add stronger before/after feedback for the patch returning to safety.
- **Player feel:** "I restored hope with my own action."
- **Difficulty:** Low.

## 4. Post-Payoff Repeatable Blessing Loop
- **Problem:** After climax, repeat play lacks a differentiated ceremonial reward.
- **Goal:** Add a lightweight repeatable blessing/prosperity beat after completion.
- **Player feel:** "The village changed because of me."
- **Difficulty:** Medium-High.
