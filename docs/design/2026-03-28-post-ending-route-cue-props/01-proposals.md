# Proposals

## 1. Active Route Cue Prop Pass
- **Problem:** Active support-point rotation is functional, but players may still rely mostly on sign text or HUD hints.
- **Goal:** Add tiny branch-colored environmental cues near the currently active point and the short route leading into it.
- **Player feel:** The village itself seems to point toward where help is flowing next.
- **Implementation difficulty:** Low.

## 2. Brother Follow-Up Reactions
- **Problem:** Heungbu and Nolbu still do not react enough to repeated post-ending labor.
- **Goal:** Add a few extra branch- and tier-aware lines on support-loop progress.
- **Player feel:** Core story figures keep acknowledging the player's continued choice.
- **Implementation difficulty:** Low.

## 3. Tier-Triggered Micro Moments
- **Problem:** Crossing landmark tiers can still be easy to miss in motion.
- **Goal:** Add a one-shot local dialogue/effect beat when the player first hits Tier 1 or Tier 2.
- **Player feel:** Milestones land with a little more ceremony.
- **Implementation difficulty:** Medium.

## 4. Post-Ending Helper Idle Animations via Props
- **Problem:** Support points and witness spots are readable, but some spaces still feel static.
- **Goal:** Fake low-cost helper presence using tiny prop states like moved baskets, open ledgers, or laid-out cloth.
- **Player feel:** The aftermath loop feels worked-in even when no one is speaking.
- **Implementation difficulty:** Medium.

## 5. Shared Summary Board Refresh
- **Problem:** The village remembers branch activity through several distributed signs, but not through one simple recurring read.
- **Goal:** Add one small summary board that updates with communal or tribute framing.
- **Player feel:** Repetition leaves a visible village memory.
- **Implementation difficulty:** Medium.
