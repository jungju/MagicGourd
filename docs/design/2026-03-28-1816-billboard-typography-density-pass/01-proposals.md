# Proposals

## Proposal A — Billboard hierarchy contract
Define distinct typography tiers for:
- spawn loop sign
- zone/lane support signs
- plaza upgrade role signs

Focus on title/body ratio, body length budget, and when a sign should read as primary vs secondary support.

## Proposal B — Plaza-only billboard quieting
Only reduce text density on the three plaza pad billboards so the current approached pad reads more clearly.

## Proposal C — Body-copy line budget + no-wrap-first rule
Keep current visual treatment mostly intact, but strictly limit body copy length and wrapping behavior so signs stop turning into text bricks.

## Proposal D — Full billboard presentation contract
Author a compact contract for:
- billboard size tier
- title/body text behavior
- always-on-top usage policy
- visual priority by sign class
- validation scenes for prompt/popup/sign overlap

This would become the final presentation layer under the existing spatial and band-separation docs.
