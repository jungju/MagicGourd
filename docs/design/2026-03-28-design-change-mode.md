# Design Change Mode

현재 `MagicGourd`는 설계 변경 모드에 들어간 상태다.

## Rule

- 설계 변경이 끝날 때까지 구현/PM/루프/메타 유지보수의 주기 실행은 스킵한다.
- 우선순위는 새 방향을 명확히 정의하는 것이다.
- 설계 변경 확정 전에는 기존 업무 진행을 최소화한다.

## Reason

현재 게임의 멀티 구조, 개인 밭 구조, 제비/호박/도깨비 루프에 대한 핵심 방향 재정의가 필요하다.

## Exit Condition

- 사용자가 새 메인 방향을 확정한다.
- 새 방향 기준 설계가 정리된다.
- PM 업무리스트가 새 방향에 맞게 재정렬된다.
- 그 다음에만 구현/루프 주기 작업을 재개한다.
