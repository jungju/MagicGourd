# Proposals

## Refine targets collected
1. **Plaza choice role clarity** — make each upgrade feel like a different pacing decision, not just a different row.
2. **Danger retreat readability** — make re-entry / retreat reads in Nolbu Area even clearer after slow.
3. **Pumpkin lifecycle legibility** — tighten the moment between plant, crack progress, and payout so owned pumpkins are even easier to read in motion.
4. **Pad approach commitment** — further polish which plaza pad feels active during approach from the center lane.
5. **Upgrade payoff afterglow** — strengthen the feeling that a purchased upgrade changes the next run immediately.

## Candidate set
### Candidate A — Plaza Choice Role Clarity
Focus on differentiating Footwork / Quick Hands / Blessing across HUD rows, pad signs, popup wording, and buy timing guidance.

### Candidate B — Pad Approach Commitment Follow-through
Focus on approach-angle and one-pad-at-a-time commitment rules for the plaza space.

### Candidate C — Upgrade Payoff Afterglow
Focus on how a buy moment hands off into the next loop so the reward feels immediately useful.

### Candidate D — Danger Retreat Readability
Focus on clearer recover / retreat / re-entry reads in the goblin slice.
