## Adoption Recommendation
Adopt **Candidate A — Plaza Choice Role Clarity**.

### Why it wins
It is the highest-value refinement that is still mostly under authoring control. The current loop already made heal / plant / break legible, but the **buy** step still asks the player to parse rows, pad signs, and purchase text as accounting data first.

This pass turns the plaza into a cleaner choice moment without expanding scope.

### Why the others were rejected for now
- **B** is valid, but the main spatial contract already exists.
- **C** is valuable, but it works better after each upgrade's role is unmistakable.
- **D** should stay attached to the blocked Studio validation bucket rather than becoming this cycle's primary authored pass.
