# Proposals

## Proposal A — Contact fairness budget
Define exact rules for when goblin pressure is allowed to feel threatening versus unfair, focusing on readable approach vectors, contact spacing, and post-tag reset windows.

## Proposal B — Danger-lane geometry cleanup
Focus mostly on static world spacing: sign, rock, and spawn-point adjustments so the lane telegraphs itself more clearly before any chase logic matters.

## Proposal C — Chase behavior polish contract
Focus on retarget cadence, leash feel, and home-return behavior so goblins read less erratically during live play without adding new mechanics.

## Proposal D — Broad danger-mode pass
Combine contact fairness, geometry, messaging, reward cadence, and route teaching into one larger pass.

## Initial read
- A is the most balanced: it can absorb small geometry/chase tuning without becoming a systems expansion.
- B is useful but too map-local; it does not fully cover chase feel after first contact.
- C is useful but too AI-local; it risks underweighting the player's spatial read.
- D is too broad for refinement mode and likely duplicates several completed passes.
