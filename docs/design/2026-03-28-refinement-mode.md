# Refinement Mode

MagicGourd는 현재 확장 모드가 아니라 **정교화 / 완성도 향상 모드**에 들어간다.

## Principle

- 새로운 큰 기능을 계속 추가하지 않는다.
- 현재 이미 잡힌 구조를 더 정교하고 완성도 높게 만든다.
- 디자인은 확장보다 정밀화에 집중한다.
- PM은 `new`보다 `rebuild / polish / verification`을 더 중시한다.
- 개발은 현재 구조의 가독성, 정합성, 체감, 연출, 안정성을 올리는 데 집중한다.

## Immediate Focus

1. house placement / orientation / scale polish
2. dialogue and HUD readability polish
3. reward and event feedback precision
4. NPC pathing / collision / interaction polish
5. quest flow consistency / cleanup / bug reduction

## Exit Condition

현재 구조가 충분히 완성도 높아졌다고 판단되기 전까지는, 큰 신규 확장보다 정교화가 우선이다.
