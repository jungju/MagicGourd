# Proposals

## 1. Rotating Support-Point Tasks
- **Problem:** Post-ending free play is readable, but too anchored to one prop per branch.
- **Goal:** Make the player walk short village loops and feel each ending inhabits more than one corner.
- **Player feel:** "The valley is still asking for care in different places."
- **Implementation difficulty:** Medium-low.

## 2. Branch Helper NPC Barks
- **Problem:** The village aftermath still feels quiet between direct interactions.
- **Goal:** Add simple ambient bark lines near branch landmarks.
- **Player feel:** The village sounds more alive.
- **Implementation difficulty:** Low.

## 3. Errand Streak Presentation Pass
- **Problem:** Repeated hand-ins could use stronger payoff punctuation.
- **Goal:** Add streak thresholds, titles, and flashier HUD callouts.
- **Player feel:** Repetition feels recognized.
- **Implementation difficulty:** Low.

## 4. Tiny Branch Route Obstacles
- **Problem:** Tribute/restoration paths do not yet feel physically different to cross.
- **Goal:** Add one or two tiny world-state blockers or aids per ending.
- **Player feel:** The village pathing itself reflects the ending.
- **Implementation difficulty:** Medium.

## 5. Ending Echo Event Pings
- **Problem:** The aftermath is static outside direct farming.
- **Goal:** Occasionally surface a branch-specific world ping or prompt event.
- **Player feel:** The ending still ripples outward.
- **Implementation difficulty:** Medium-high.
