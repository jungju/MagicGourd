# Proposals

## Proposal A — Nearest Upgrade Threshold Contract
Define one canonical "nearest meaningful spend" rule so the HUD, payout copy, and plaza support text all point at the same next threshold.

## Proposal B — Pumpkins-To-Buy Readability Pass
Translate money progress into `one more pumpkin` / `two more pumpkins` style phrasing wherever it helps.

## Proposal C — Cheapest Affordable Highlight Contract
When the player can afford more than one upgrade, define a consistent way to highlight the cheapest or most immediate buy without turning plaza choice into noise.

## Proposal D — Spend Window Soft-Prompting
Author a small rule set for when the loop suggests returning to plaza now versus finishing the current safe action first.
