- pumpkin payouts happen often
- plaza upgrades are core progression

The refinement is not about inventing more content.
It is about making existing progress feel more authored.

## Fun check
Passing this design should make the player feel a cleaner mini-arc:
1. earn money
2. feel the threshold approaching
3. unlock a spend window
4. choose whether to finish the local beat or return to plaza
5. feel the next run change immediately

That is a stronger arcade rhythm than `number goes up until I remember the plaza exists`.
