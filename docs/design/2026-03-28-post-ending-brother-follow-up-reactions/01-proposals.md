# Proposals

## 1. Brother Follow-Up Reactions
- **Problem:** Heungbu and Nolbu do not react enough to repeated post-ending labor.
- **Goal:** Add branch-aware, tier-aware, low-spam follow-up lines when the player revisits them in `COMPLETE`.
- **Player feel:** The ending choice still matters to the core relationship, not just to props and signs.
- **Implementation difficulty:** Low.

## 2. Shared Summary Board Refresh
- **Problem:** The village still lacks one simple recap object that reflects replay-loop growth.
- **Goal:** Add one board/sign that updates with branch-specific summary language.
- **Player feel:** The village remembers what has been happening.
- **Implementation difficulty:** Medium.

## 3. Tier-Triggered Micro Moments
- **Problem:** Landmark tier gains can pass quietly if the player is focused on errands.
- **Goal:** Add one-shot milestone beats the first time Tier 1 and Tier 2 are reached.
- **Player feel:** Repetition lands with a bit more ceremony.
- **Implementation difficulty:** Medium.

## 4. End-Branch Ambient Duet Lines
- **Problem:** Post-ending ambient dialogue is present, but the brothers still mostly speak as isolated actors.
- **Goal:** Add paired follow-up exchanges where one brother's line implicitly answers the village state created by the other.
- **Player feel:** The branch aftermath feels more narratively interconnected.
- **Implementation difficulty:** Medium.

## 5. Support-Point Helper Idle States
- **Problem:** Some support points still rely mostly on signs and prompts between interactions.
- **Goal:** Add small changed prop states suggesting recent help, tallying, or relief work.
- **Player feel:** The village looks recently used.
- **Implementation difficulty:** Medium.
