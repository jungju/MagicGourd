# Candidate Set

## Candidate 1 - Final Audience Choice
- Core loop: clear brambles -> speak with Heungbu -> attend village decision beat -> choose kindness/restoration or tribute/fear response -> receive ending.
- Theme role: puts the folktale's moral contrast directly in the player's hands.
- Expected payoff: clear authored ending, strong replay value, simple implementation path.

## Candidate 2 - Hidden Moral Score Ending
- Core loop: finish story -> game evaluates prior helpful vs greedy actions -> auto-assign ending when speaking to Heungbu.
- Theme role: frames the village as judging your accumulated conduct rather than one explicit choice.
- Expected payoff: more literary tone, lighter UI burden, but lower clarity.

## Candidate 3 - Ceremony Plus Timed Recovery Ending
- Core loop: clear brambles -> trigger village festival prep -> finish one last restoration action under light pressure -> receive hopeful or diminished ending.
- Theme role: emphasizes rebuilding after envy and gives the finale a more physical task.
- Expected payoff: more dramatic conclusion, but needs more new content than the others.

## Candidate 4 - Epilogue Reputation Loop
- Core loop: enter free-play epilogue -> perform repeated deliveries/favors -> eventually unlock a prosperity or greed ending.
- Theme role: treats endings as the long shadow of village habits.
- Expected payoff: deeper progression runway, but too broad for the current slice.
