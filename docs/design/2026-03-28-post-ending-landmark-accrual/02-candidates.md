# Candidate Set

## Candidate 1
- Core loop: Repeat post-ending errands and watch nearby landmarks visually accumulate branch-specific support/tension.
- Theme role: Makes the ending visible in the village fabric, not only in dialogue.
- Expected payoff: Strong at-a-glance progression with low systemic risk.

## Candidate 2
- Core loop: Visit active support points to hear new helper barks and short reactions.
- Theme role: Turns endings into local social texture.
- Expected payoff: Cheap warmth/flavor, but weaker gameplay presence.

## Candidate 3
- Core loop: Trigger occasional rare branch-specific events after enough errands.
- Theme role: Makes the aftermath feel alive and less predictable.
- Expected payoff: High burst value, but more timing/state complexity than the current layer needs.

## Candidate 4
- Core loop: Follow support-point rotations through route markers and path cues.
- Theme role: Makes restoration/tribute movement patterns feel spatially authored.
- Expected payoff: Nice movement polish, but less persistent payoff than landmark accumulation.
