# Proposals

## Proposal A
- Summary: Add branch-specific landmark accrual tiers at post-ending support points.
- Why it helps: Gives replay visible payoff using counters and props the project already supports.
- Cost: Moderate.
- Risk: Can become noisy if too many props change at once.

## Proposal B
- Summary: Add helper NPC barks that appear at active support points after repeat errands.
- Why it helps: Cheap narrative reinforcement with little world-authoring overhead.
- Cost: Low.
- Risk: Mostly text value; may feel thin after the recent ambient dialogue expansion.

## Proposal C
- Summary: Add rare branch-specific micro-events every few errands (seed gift, inspection flare, villager comment pulse).
- Why it helps: Creates surprise and cadence in free play.
- Cost: Moderate to high.
- Risk: Hidden cadence logic and event timing can feel arbitrary before the base replay layer fully settles.

## Proposal D
- Summary: Add route props between support points so movement itself feels different by ending.
- Why it helps: Strengthens support-point rotation spatially.
- Cost: Moderate.
- Risk: Easy to overbuild into map dressing without enough gameplay gain.
