# Candidate Set

## Candidate 1 - Compact Retaliation Pressure Loop
- Core loop: inspect sabotage -> clear brambles -> stabilize one village support point -> return to Heungbu.
- Theme role: shows greed as disruptive pressure and kindness as active repair.
- Expected payoff: stronger bridge between miracle payoff and final audience.

## Candidate 2 - Retaliation Patrol Lockdown
- Core loop: avoid or wait out heightened Nolbu patrol pressure before restoring the patch.
- Theme role: makes Nolbu's control more physical and intimidating.
- Expected payoff: higher tension and stronger estate presence.

## Candidate 3 - Communal Recovery Marker System
- Core loop: after sabotage, restore 2-3 lightweight village markers such as lantern, basket, or sign.
- Theme role: emphasizes the valley healing together.
- Expected payoff: visible environmental repair and stronger setup for restoration ending.

## Candidate 4 - Dialogue-Heavy Retaliation Reframe
- Core loop: mostly existing bramble clear flow, but with richer staged dialogue/sign changes.
- Theme role: reframes retaliation through authored tone rather than new mechanics.
- Expected payoff: cheap improvement, limited systemic gain.
