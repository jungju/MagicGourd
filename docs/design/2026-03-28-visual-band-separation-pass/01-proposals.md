# Proposals

## Refine Targets Collected
1. Spawn/plaza/village signboards sometimes risk sharing the same read band as prompts.
2. Plaza purchase popup readability can still be weakened by billboard density even after prompt-distance tightening.
3. Pumpkin payout and crack-progress popups need protected headroom that is distinct from support signage.
4. Danger-lane warning signs should support anticipation without sitting in the same band as goblin contact/popup reads.
5. The project has readability budgets, but not a single camera-layer rule that authors can apply before adding or moving props.

## Proposal A — Visual Band Separation Contract
Define explicit vertical/readability roles for:
- decision band (prompt)
- feedback band (world popup)
- support band (billboard)

Use that as the tie-breaker for small map and copy changes.

## Proposal B — Plaza Sign Compression Pass
Focus only on plaza signs/pads so each upgrade choice reads clearly and the purchase popup keeps room.

## Proposal C — Farming Headroom Pass
Focus on swallows/pumpkins and ensure reward/progress popups do not collide with lane signage.

## Proposal D — Danger Warning Framing Pass
Focus on danger entry, keeping warning signs and goblin silhouettes readable without overlap or late contact.
