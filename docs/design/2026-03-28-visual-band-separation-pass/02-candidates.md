# Candidates

## Candidate A — Whole-map visual band separation contract
A compact rule set for how prompts, world popups, and billboards occupy different read bands across the whole arcade loop.

### Benefits
- Helps every zone at once
- Gives future tuning a shared tie-breaker
- Directly supports existing spatial/cadence/readability docs

### Risks
- Must stay concise so it does not duplicate prior documents

---

## Candidate B — Plaza-only sign/popup clearance pass
Narrow the work to central plaza pad signs, prompt lanes, and purchase popup headroom.

### Benefits
- High-value local payoff
- Easy to validate quickly

### Risks
- Too local; does not solve village/pumpkin/danger slices
- Can become a subset of a broader rule set

---

## Candidate C — Farming popup headroom contract
Focus on swallow and pumpkin interactions so heal/plant/crack/payout feedback always reads cleanly.

### Benefits
- Strong reward-feel impact
- Protects the core loop cadence

### Risks
- Misses plaza and danger overlap cases
- Duplicates part of the cadence/spatial docs

---

## Candidate D — Danger warning layering contract
Focus on ensuring goblin anticipation cues, signs, and slow feedback do not share the same clutter band.

### Benefits
- Improves fairness feel
- Supports MG-PM-408 validation

### Risks
- Useful, but narrower than the actual cross-map completion gap
