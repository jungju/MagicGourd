# Proposals

## Refine Targets Collected
1. **Action verb coherence** across swallow / seed / pumpkin / upgrade interactions.
2. **Zone sign readability** so plaza, village, and danger zone each describe one purpose cleanly.
3. **Feedback bucket precision** so success/warn/info messages sound distinct and rewarding.
4. **Upgrade pad naming clarity** so `Upgrade` becomes a purchase/read decision instead of a generic system word.
5. **Button and prompt agreement** so the bottom action button teaches the same verb the prompt later reinforces.
6. **World popup payoff language** so floating text reinforces gain/loss clearly.

## Compressed Candidate Set

### Candidate A — Verb Map Unification
Define one canonical verb set for all loop actions and apply it to prompts, button copy, HUD guidance, and feedback.

### Candidate B — Zone Sign + HUD Support Text Tightening
Rewrite world billboards and low-priority HUD support text so each zone advertises a single purpose in fewer words.

### Candidate C — Reward / Warning Feedback Taxonomy
Refine banner and popup copy into clear emotional buckets: kindness gain, harvest payout, purchase confirmation, danger interruption, recovery.

### Candidate D — Combined Loop Copy Coherence Pass
Ship a single copy system covering verbs, signs, button text, prompt text, feedback buckets, and short world popup phrases without adding new mechanics.
